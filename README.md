1. 实时订阅 odom + imu 经ekf松耦合后的局部里程计数据；
2. 订阅amcl计算出的全局位姿话题；
3. 对大协方差和小协方差但大位姿、大角度偏转、大顺时角速度的位姿进行过滤；
4. 按时间戳对齐将过滤后的amcl的位姿用于校正odom位姿，使其成为全局位姿；
5. 本算法为自动识别，自动触发校正；
6. 本算法订阅机器人急停话题以及人工介入、切图服务，一旦触发，则停止内部计算，并清空数据、将状态切换至初始状态。

P.S.*本算法不发布tf树变换、不在变换之后进行插值操作，只在外围充当amcl的位姿校正功能。 
若要将本算法融合进全局定位底层，则需要剥夺amcl或其他定位算法的tf树发布控制权，本算法通过订阅定位算法位姿来统一发布全局位姿和odom->map的tf树变换。
